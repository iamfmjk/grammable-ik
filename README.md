# Grammable

Photo feed web application built in TDD style as a part of Berkeley Coding Bootcamp

**Built with:** Ruby on Rails, RSpec, Bootstrap 4, Devise, Carrierwave, Simple_form

## Features

- Browse images
- Post comments
- Post images with description
